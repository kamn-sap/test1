# Frontend Placeholder

This directory will contain the React frontend for the test case management app.

## Setup

You can bootstrap a React application here with Create React App or your preferred framework:

```bash
npx create-react-app . --template typescript
```

After generating, start the dev server with:

```bash
npm install
npm start
```

The app should interact with the backend services defined in `../infra` (AppSync, Cognito, etc.).
